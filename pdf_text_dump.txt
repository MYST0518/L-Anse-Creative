1. なぜ、いま 「 お 店 の 歌 （ 公 式 ソング ）」 が 必 要 なのか ？
最 近 、スマートフォンで 動画（ YouTube や TikTok など ） を 見 る 人 が 急 増 しています。こうしたネット 広 告 やお
店 の 紹 介 動画 では、 映 像 と 同 じくらい「 耳 に 残 る 音 楽 やリズム 」 がとても 重 要 になっています。
文 字 や 画 像 だけの 広 告 は 見 逃 されやすいですが、お 店 の 特 長 や 名 前 が 入 った 楽 しいメロディがあると、 一 瞬 で
「 あ、あのお 店 の 曲 だ ！」 と 思 い 出 してもらえるようになり、 他 のお 店 との 大 きな 違 いを 作 ることができま
す。
今 回 は、プロに 歌 を 注 文 するのではなく、ネット 上 の 多 くの 音 楽 クリエイターに 「 曲 を 作 ってもらうコンテス
ト 」 を 提案 いたします。
💡  ファンコミュニティが 育 つ 仕 組 み
コンテストを 開 くと、 応 募 するクリエイターたちが 「 いま 〇〇 店 のための 曲 を 作 っているよ 」 と、 制 作 過 程
や 完 成 した 楽 曲 をご 自 身 の SNS に 楽 しんで 投 稿 してくれます。これにより、 従 来 の 広 告 とは 異 なり、クリエ
イターやそのリスナーが 「 お 店 のサポーター （ ファン ）」 として 自 然 に 巻 き 込 まれていく 温 かい 広 がりが 生 ま
れます。
2. どんなイベント ？ （ 企 画 の 概 要 ）
ネット 上 で 歌 やボカロ 曲 、 AI ミュージックを 作 っている 全 国 のクリエイターへ 向 けて、お 店 の 魅 力 を 伝 える 明
るくキャッチーな 「 公 式 ソング 」 を 募 集 します。
募 集 するもの [ お 店 の 名 前 ] が 入 ったイメージソング
（動画 で 使 いやすいよう、 15 秒 〜 30 秒 程 度 の 短 いサビだけの 応 募 も OK です ）
募 集 する 期 間 募 集 を 開 始 してから 約 1 ヶ 月 間
結 果 の 発表 締 め 切 りから 約 2 週 間 後（ ホームページや SNS で 発表 します ）
選 び 方
事 務 局 で 「一 次 審 査 （ スクリーニング ： 音 質 チェックや 応 募 規 約 の 適 合 確認 ）」
を 行 ったうえで、 厳 選 した 候 補曲 をオーナー 様 へご 提 示 します。オーナー 様 は 最
終 候 補 の 中 からお 気 に 入 りの 1 曲 を 選 ぶだけなので、 選 考 負担 は 最 小 限 です。

3. コンテストを 開 くことで、お 店 が 得 られる 3 つのメリット
①  クリエイターとのつながりが 生 む 「 応 援 の 輪 」
参 加 するクリエイターは 「 お 店 を 応 援 したい 」「 自 分 の 作品 を 多 くの 人 に 聴 いてほしい 」 という 前 向 きな 想 い
から、 自 ら SNS 等 で 楽 曲 を 紹 介 します。これにより、 単 なる 広 告 の 配 信 ではなく、クリエイターのフォロワー
へお 店 の 魅 力 が 好 意的 に 伝 わっていきます。
②  クリエイターのファンがお 店 のファンになる 親 近 感
「 自 分 の 好 きなクリエイターがお 店 の 曲 を 作 っている 」 という 過 程 を 見 ることで、ファンはお 店 に 対 しても 親
近 感 を 持 つようになります。 通 常 の 広 告 よりも 良 い 印 象 を 持 たれやすく、ファンがそのままお 店 のお 客 様 にな
ってくれる 可 能 性 が 高 まります。
③  二 次 利用 も 自 由 ！ ずっと 無 償 で 使 える 「利用 許 諾 」 形 式
コンテストで 採 用 された 楽 曲 の著 作 権 および 原 盤 権 はアーティスト （ クリエイター ） に 帰 属したまま、 店 舗 側
は「無 期 限 かつ 原 則 として 取 消 不 能 な 利用 許 諾 （ ライセンス ）」を 取 得 します。これにより、 後 々 の 追 加 支 払 い
などを 気 にせず、 SNS 動画 BGM 、 店 内 BGM 、ローカル CM などの 広 告 、チラシの QR コード 誘 導 など、店 舗 プロ
モ ーションに 関 わる 用 途 であればすべて 無 制 限 ‧ 永 久 無 償 でご 利用 いただけます。動画 の 長 さに 合 わせたトリ
ミング 等 の 改 変 （ カット‧フェードアウト 等 ） も 許 諾 され、 余 計 な 手 間 をかけずに 活 用 できます。
おすすめの 使 い 道 詳細 とメリット
SNS 動画 の BGM に お 店 の TikTok や Instagram 、 YouTube にのせる 動画 の 背 景 音 楽 として 使 えま
す （見 逃 し 防 止 に 効 果 的 ）
店 内 の BGM に お 店 オリジナルの BGM として 流 すことで、 他 店 にはない 心 地 よい 雰 囲 気 作
りと 特 別 な 空 間 を 演 出 できます
CM やチラシの QR コードに ローカル CM のジングルとして 使 ったり、チラシに 曲 の QR コードを 載 せて 聴
いてもらうことができます

A 利用 許 諾 形 式 のため、 金 銭 負担 を 抑 えつつ、お 食 事 券 などを 活 用 した 「 アンバサダー （ ファン ）

著 作 権 の 完 全 譲 渡 を 強 いるコンテストは、 高 額 な 賞 金 がないと 買 い 叩 き （ クリエイター 搾 取 ）

Q コンテスト 採 用 曲 の 権 利関 係 （ 著 作 権 ‧ 原 盤 権 ‧ 配 信 ） や 将 来 的 なリスク
管 理 はどうなっていますか ？
A クリエイターの 権 利 を 尊 重 しつつ、 店 舗 側 が 永 久 的 に 無 償 利用 できる 取 消 不 能 ライセン
スを 提 供 し、トラブルのない 仕 組 みを 構 築 しています。
権 利 の 帰 属 と 利用 許 諾 ： 採 用 作品 の 著 作 権 ‧ 原 盤 権 はアーティストに 帰 属 しま
す。 店 舗 側 には、 将 来 的 に 取 り 消 しや 解 除 ができない「無 期 限 かつ 原 則 として 取
消 不 能 なプロ モ ーション 利用 権 （ ライセンス ）」が 付 与 されます。 店 舗 PR に 関 わ
る 用 途 （ SNS 動画 BGM 、 店 内 BGM 、 広 告 、チラシ QR など ） であれば 制 限 なく 利
用 可 能 です。 動画 の 長 さに 合 わせたトリミング 等 の 改 変 も 許 諾 されます。
お 申 し 込 み 後 の 配 信代 行 ： 採 用 楽 曲 のサブスク 配 信 サービスへの 配 信 ‧ 流 通
は、 原 則 として 株式会社 Coedo Music Labo が 代 行 します。 配 信 から 発 生 した 収
益 は、アーティストと 株式会社 Coedo Music Labo で 「 5:5 」 の 比 率 で 折 半 ‧ 分 配
します。 応 募 者 はエントリー 時 点 で 本 配 信代 行 と 収 益 分 配 に 同 意 する 形 となり、
採 用 後 に 配 信 を 拒 否 することは 規 約 上 できません （ 拒 否 した 場合 は 規 約 違 反 とし
て 採 用 および 賞 品 が 取 り 消 されます ） 。
閉 店 ‧ 事 業 譲 渡 ‧ 運営 者 変更 時 の 対 応 ： 店 舗 の 閉 店 、 事 業 譲 渡 、 運営 法 人 の 変
更 等 に 伴 う 楽 曲 利用 権 の 引 き 継 ぎについては、 個 別 にご 相 談 の 上 でスムーズに 対
応 いたします。
株式会社 Coedo Music Labo の 事 業 終 了 時 ： 万 が 一 、 株式会社 Coedo Music
Labo の 事 業 終 了 等 の 事 由 が 発 生 した 場合 でも、 店 舗 側 のプロ モ ーション 利用 権
はそのまま 維 持 されます。 事 業 終 了 の 90 日 前 までに 通 達 の 上 、 配 信 を 停 止 し、 諸
権 利 をアーティストまたは 店 舗 側 のいずれかに 譲 渡 するための 三 者 間 協 議 を 行 い
ます。

Q Suno AI や Udio などの 生 成 AI を 用 いた 楽 曲 の 応 募 は 可 能 ですか ？ また、 著
作 権 トラブルへの 対 策 は ？
A 生 成 AI 楽 曲 の 応 募 は 「 可 」 とします。 規 約 とチェックボックスによる 厳 格 な 同 意 形 成 に
より、 店 舗 側 にトラブルが 及 ばない 防 衛 線 を 張 っています。
AI 楽 曲 の 歓 迎 ： 生 成 AI （ Suno AI 、 Udio 等 ） や 歌 声 合 成 ソフト （ 初 音 ミク 等 ） を
使 用 した 作品 も 広 く 歓 迎 します。 応 募 時 は 「 AI 使 用  of 無 い 」 および 「 使 用 した
AI サービス 名 」 の 申 告 を 必 須 化 します。
AI の 商 用利用 規 約 遵 守 ： 応 募 時 に 「 商 用利用 が 認 められている 有 料 プランで 作
成 ‧ 出 力 したこと 」「 AI サービス 側 の 利用 規 約 を 完 全 に 遵 守 していること 」 への
同 意 （ チェックボックス ） を 必 須 とし、 規 約 遵 守 の 責 任 を 応 募 者 本人 に 帰 属 させ
ます。
第 三 者 権 利 侵 害 時 の 責 任 （ 防 衛 線 ） ： 応 募 作品 が 「 アーティスト 自 身 の 完 全 な
オリジナル 未 発表作品 であること 」 を 保 証 させ、 万 が 一 、 他者 の 著 作 権 侵 害 や 権
利 を 侵 害 する AI 生 成 等 により 紛 争 が 生 じた 場合 は、アーティスト 自 身 の 責 任 と 費
用 負担 において 解 決 するものとします。 店 舗 および 株式会社 Coedo Music Labo
は 一 切 の 責 任 を 負 わない 防 衛 線 を 敷 きます。

Q 応 募 数 はどのくらい 想 定 されますか ？ また、 応 募 が 少 数 ‧またはゼロだっ
た 場合 の 対 応 や、 審 査 基 準 は ？
A 1 回 のコンテストあたり 15 〜 30 曲 前 後 を 想 定 しています。 応 募 状 況 に 応 じた 安心 の 救 済
措 置 や、 明 確 な 審 査 基 準 を 用 意 しています。
過 去 実 績 ： 神 戸 のカレー 名 店 「 マンドリルカレー 」 様 で 4.1 万 回 表 示 、 Record
Cafe Time 様 で 1.9 万 回 表 示 の SNS インプレッション‧ 拡散 実 績 があります。
応 募 少 数 の 場合 ： 応 募 数 が 少 数 であっても、 原 則 としてその 中 から 厳 正 に 選 考 を
行 い、 最 優 秀 作品 （ 採 用） を 決 定 します。 締 切 直 前 に 少 ない 場合 は、 miya の
SNS ネットワーク 等 を 活 用 して 追 加 の 告 知 ブーストを 行 います。
応 募 ゼロの 場合（ リスクゼロ 設 計 ） ： 規 約 に 「 審 査 基 準 に 満 たない 場合 は 該 当
作品 なしとする 場合 がある 」 と 明 記 して 防 衛 線 を 張 るほか、 万 が 一 の 救 済 措 置 と
して、 「 募 集 期 間 の 延 長 」 、またはコンテスト 費 用 をそのままスライドして 「株式
会社 Coedo Music Labo によるダイレクト 楽 曲 制 作 ‧ 納 品 プラン 」 へ 切 り 替 える
ことで、 店 舗 側 が 『 お 金 を 払 ったのに 曲 が 手 に 入 らない 』 というリスクをゼロに
します。
審 査 基 準 の 明 文 化 ： 「 ①  店 舗 コンセプトとの 親 和 性 （ 必 須 の 言 葉 や 雰 囲 気 の 反
映 ）」「 ②  楽 曲 のキャッチーさ‧ 実 用 性 」「 ③  オリジナリティと 熱 量 」 の 3 軸 を 明
記 します。また、 選 考 結 果 の 理 由 に 関 する 問 い 合 わせには 一 切 応 じない 免 責 事 項
も 記 載 します。
結 果 発表 の 形 式 ： YouTube 、 TikTok 、ポッドキャストなどの 「 ネット 配 信 （生
配 信 またはアーカイブ 配 信 ）」 による 特 番 形 式 で 発表 します。 生 配 信 ならではの
熱 量 あるイベントバズを 生 み 出 し、アーカイブとしてお 店 の 資 産 に 残 るメリット
を 提 供 します。

9. 届 いた 曲 をスマホやパソコンでパッと 聴 ける 「 かんたん 審
査 画 面 」
オーナー 様 には、 忙 しい 業 務 の 合間 に 手 軽 に 応 募 曲 をチェックできる、 専 用 の 「 かんたん 審 査 画 面 」 をご 用 意
します。
クリエイターが 送信 した 音 源 ファイルや 歌 詞 テキストが、ダッシュボード 上 に 自 動 で 整 理 されて 並 ぶため、フ
ァイルを 1 つずつダウンロードして 開 くといった 面 倒 な 作業 は 一 切 不 要 です。
クリエイターが応募する 事務局がデータ整理‧⼀次選考 オーナー様へ最終候補を推薦 スマホや PC でパッと試聴 合格‧保留を選ぶだけ
できること 詳細
その 場 でワンタップ 再 生 音 楽 プレイヤーが 画 面 についているので、 再 生 ボタンを 押 すだけでその 場
で 曲 を 聴 くことができます。
歌 詞 や 自 己 PR を 同 時 確認 クリエイターが 曲 に 込 めた 想 いや、 書 いた 歌 詞 がプレイヤーのすぐ 横 に 表
示 され、 一 緒 に 確認 できます。
直 感 的 なお 気 に 入 り 整 理 「 お 気 に 入 り 」「 保 留 」「 選 外 」 などのステータスボタンをぽちぽち 押 すだ
けで、 直 感 的 に 楽 曲 を 仕 分 けられます。
